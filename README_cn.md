# PDTricker 介绍
PDTricker 是MuseLab设计的PD诱骗工具，使用沁恒的ch224k和ch552t实现，支持PD3.0/2.0，可通过按键配置输出5V/9V/12V/15V/20V， 硬件原理图开源， 软件源码开源，您也可自行修改源码定制实现个性化需求。
